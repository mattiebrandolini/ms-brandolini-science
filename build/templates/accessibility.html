Accessibility

How this site is built to work for everyone.

Our commitment

This site serves students with varying levels of English proficiency, different learning needs, and different devices. Accessibility is not an afterthought — it's built into every page. We target WCAG 2.2 Level AA conformance.

Built-in features

Use the ⚙️ button in the top navigation bar to access these settings. All choices are saved automatically.

  • Dark / light theme — choose whichever is easier on your eyes
  • Dyslexia-friendly font — switches all text to OpenDyslexic
  • Colorblind-friendly mode — replaces color-only indicators with shapes and borders
  • Extra spacing — increases line height, letter spacing, and word spacing
  • Reduced motion — disables all animations (also auto-detects your device setting)
  • Text size — small, medium, or large base font

Technical details

  • All pages have a skip-to-content link for keyboard and screen reader users
  • All pages use semantic HTML with proper landmarks (<main>, <nav>)
  • All fonts are self-hosted — no external requests that could be blocked by school networks
  • Resource pages have QR codes so students can access content on their phones when Chromebook filters block links
  • Resource data has an offline fallback — pages work even when the internet is slow or filtered
  • A print stylesheet is included on every page for clean paper output

Known limitations

  • Some linked resources (YouTube videos, external simulations) are hosted by third parties and may not meet accessibility standards
  • School content filters may block specific resources unpredictably — QR codes and the "Report a broken link" feature help work around this
  • The resource viewer requires JavaScript to display interactively (a direct link to the data is provided when JS is unavailable)

Report a barrier

If something on this site doesn't work for you, please let us know:

  • Report an accessibility issue on GitHub
  • Or tell Ms. Brandolini directly — in person, by email, or through your teacher
